# Proyecto: Inseguridad Alimentaria en Lima Metropolitana

Aplicación Streamlit con colores y animaciones para predecir y clasificar el riesgo alimentario por distrito.

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Despliegue en Streamlit Cloud

1. Sube estos archivos a GitHub.
2. En Streamlit Community Cloud selecciona el repositorio.
3. Main file path: `app.py`.
4. Presiona Deploy.
